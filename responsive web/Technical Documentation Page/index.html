Introduction

JavaScript is a versatile programming language commonly used in web development.

It allows for dynamic content, making webpages interactive and responsive to user actions.

JavaScript can run both in the browser and on the server side using environments such as Node.js.

console.log("Hello, world!");
Variables

Variables in JavaScript are used to store data values.

You can declare variables using var, let, or const.

let x = 10; const y = 20;
  • var is function-scoped.
  • let and const are block-scoped.
Functions

Functions allow you to encapsulate code and reuse it.

You can define a function using the function keyword.

function greet(name) { return "Hello " + name; }
  • Functions can take arguments.
  • They can return a value.
Objects

Objects in JavaScript are collections of key-value pairs.

They allow you to group related data and functions.

const person = { name: "John", age: 30 };
  • Objects can have properties.
  • They can also have methods (functions).
Arrays

Arrays are used to store multiple values in a single variable.

Each value in an array has a numerical index, starting at 0.

const fruits = ["apple", "banana", "cherry"];
  • Arrays can hold any type of data.
  • They are zero-indexed.
